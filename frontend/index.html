    Welcome

    Use Tools to run either pipeline; they do not share outputs.

    Direct3D-S2 — turns one image into a watertight 3D mesh with a neural sparse-volume model. GitHub

    PIXEstL — slices an image into filament-colored layers and exports stacked STLs and PNG previews for multi-material printing. GitHub

    Direct3D-S2

    Image → OBJ via Direct3D-S2. Use the sidebar for options, then Tools → Run or Generate 3D (D3D).

    D3D mesh library

    Select a mesh to preview in the 3D view (below).

    PIXEstL

    Image → PNG previews (quantized color + texture). Export ZIP for STL outputs. Python PIXEstL port (not Direct3D-S2).

    Source image

    Run Generate or Refresh to load previews.

    Server hosts two tool APIs: /api/v1/d3d/* and /api/v1/pixestl/*.
